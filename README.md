# Udacity Blockchain Capstone

## How to test contracts locally?
1. Download this repo and unzip it to the folder
2. Open the Terminal at this folder
3. Install dependencies
```bash
npm install
```
4. Compile contracts
```bash
truffle compile
```
5. Run automated tests
```bash
truffle test
```
